📚 About This Project

This is an educational demo of a hospital management website, built to teach students how to work with web technologies — HTML, CSS, JavaScript, PHP, and SQLite.

The site models Mulago National Referral Hospital, Uganda's largest public hospital. All patient data shown is fictional. No real PHI (Protected Health Information) is stored or transmitted.

Key concepts demonstrated: dynamic HTML rendering · localStorage CRUD · form validation · responsive design · admin authentication · CSV export · live status computation from time.
